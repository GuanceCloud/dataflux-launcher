# DataFlux Launcher 

DataFlux Launcher 是一个 [观测云](https://guance.com) 私有部署版本的版本管理工具与部署、运维工具。

本工具使用最基本的 Web 技术方案实现：jQuery + Bootstrap + Jinja2。
